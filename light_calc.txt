Data Sources That Have Real Numbers
1. Academic Horticulture Databases
Plants For A Future (PFAF) Database

https://pfaf.org/
Has some quantitative data
Downloadable database available
Example: "Light: 3-5" (scale of 1-5, where 5 = full sun)
Free, scrape-able

But limitation: Still somewhat qualitative

2. Research Papers (Most Accurate)
Google Scholar / ScienceDirect
Search for: "[plant name]" AND "photosynthetic photon flux" AND "optimal"
Example real data from papers:
Lettuce (Lactuca sativa):
- Optimal PPFD: 200-400 μmol/m²/s
- Maximum tolerable: 600 μmol/m²/s
- Photoperiod: 12-16 hours (longer causes bolting)
- Temperature interaction: >25°C + long days = bolting

Tomato (Solanum lycopersicum):
- Optimal PPFD: 400-600 μmol/m²/s  
- Maximum: 1000 μmol/m²/s
- Photoperiod: Day-neutral (doesn't care about length)
- UV tolerance: High
Problems:

Manual collection (no API)
Not available for all plants
Paywalls


3. TRY Plant Trait Database
What it has:

280,000+ species
Quantitative traits including:

Light compensation point
Light saturation point
Shade tolerance scores



URL: https://www.try-db.org/
Access:

Free for non-commercial research
Need to request data (not instant API)
Commercial use requires permission

This might be your best bet for quantitative data.

4. USDA ARS (Agricultural Research Service)
GRIN-Global (Germplasm Resources Information Network)

https://npgsweb.ars-grin.gov/
More detailed than Plants Database
Sometimes has research notes with numbers


Realistic Hybrid Approach
Since comprehensive quantitative data doesn't exist for all plants, build a tiered system:
Tier 1: Plants with Research Data (High Confidence)
json{
  "symbol": "LASA",
  "commonName": "Lettuce",
  "light": {
    "optimal_ppfd": [200, 400],  // μmol/m²/s
    "max_ppfd": 600,
    "photoperiod_max": 16,  // hours before bolting
    "source": "doi:10.1234/hortsci.2020.12345"
  }
}
Tier 2: Plants with Inferred Data (Medium Confidence)
json{
  "symbol": "ABAM",
  "commonName": "Pacific silver fir",
  "light": {
    "usda_description": "Full sun to partial shade",
    "inferred_min_hours": 4,
    "inferred_max_hours": null,  // No upper limit known
    "confidence": "medium"
  }
}
Tier 3: Vague USDA Data Only (Low Confidence)
json{
  "symbol": "UNKN",
  "light": {
    "usda_description": "Full sun",
    "confidence": "low"
  }
}

Other Traits to Scrape While You're There
Since you're already parsing papers, look for these patterns too:
1. Temperature Requirements
pythonpatterns = {
    'optimal_temp': r'optimal.*?(\d+)[-–](\d+)?\s*[°◦]?C',
    'min_temp': r'minimum.*?(\d+)\s*[°◦]?C',
    'max_temp': r'maximum.*?(\d+)\s*[°◦]?C',
    'frost_tolerance': r'frost.{0,20}(tolerant|sensitive|hardy)',
}
Example matches:

"Optimal growth at 20-25°C"
"Minimum temperature 5°C"
"Frost sensitive below 0°C"


2. Water Requirements
pythonpatterns = {
    'water_need': r'water.{0,30}(\d+)\s*mm',
    'drought_tolerance': r'drought.{0,20}(tolerant|sensitive|resistant)',
    'irrigation': r'irrigat(ion|ed).{0,30}(\d+)\s*(mm|L)',
}

3. pH Tolerance
pythonpatterns = {
    'optimal_ph': r'pH\s*(\d+\.?\d*)[-–](\d+\.?\d*)',
    'ph_range': r'pH.{0,20}(\d+\.?\d*).{0,10}(\d+\.?\d*)',
}
Example: "Plants grew best at pH 6.5-7.0"

4. Nutrient Requirements
pythonpatterns = {
    'nitrogen': r'nitrogen.{0,30}(\d+)\s*(ppm|mg/L|kg/ha)',
    'npk_ratio': r'(\d+)[-:](\d+)[-:](\d+)',  # Like "10-10-10"
}

5. Growth Characteristics
pythonpatterns = {
    'days_to_maturity': r'(\d+)\s*days?.{0,20}(maturity|harvest)',
    'plant_height': r'height.{0,20}(\d+)[-–]?(\d+)?\s*(cm|m)',
    'growth_rate': r'growth rate.{0,30}(slow|medium|fast|rapid)',
}

6. Photoperiod (You mentioned this)
pythonpatterns = {
    'photoperiod_type': r'(short|long|day-neutral).{0,10}(day|photoperiod)',
    'critical_daylength': r'critical.{0,20}(\d+)\s*h(our)?s?',
    'bolting': r'bolt(ing|ed).{0,30}(\d+)\s*h(our)?s?',
}

7. Disease/Pest Resistance
pythonpatterns = {
    'disease_resistant': r'resistant to (.+?)(?:;|\.|\,)',
    'disease_susceptible': r'susceptible to (.+?)(?:;|\.|\,)',
}

8. Soil Preferences
pythonpatterns = {
    'soil_type': r'(sandy|clay|loam|peat).{0,10}soil',
    'drainage': r'(well-drained|poorly-drained|waterlogged)',
    'organic_matter': r'organic matter.{0,30}(\d+)',
}
